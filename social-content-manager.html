About Comments & Auto-Reply
This app does not auto-reply to Facebook comments or DMs. Doing that safely requires a server running continuously to watch for new messages — not something a browser-only app can do, and risky for B2B trust if a bot replies with the wrong info. Instead, the Dashboard's Inquiry & Comment Check reminds you which published posts to look at on Facebook directly, so real inquiries get a real reply from you.
Default Posting Times
Used by Auto-Generate Week to suggest a posting time per weekday based on the content pillar rotation.
Full Backup
Saves everything — posts, photos, brand profiles, Facebook page settings, and API keys — into one file you can restore later or move to another computer.
Restoring replaces all current data in this browser. Back up first if unsure.
Spreadsheet Export
Quick CSV of all posts for review in Excel or Google Sheets — not a full backup.
Danger Zone
This permanently deletes everything in this browser. Download a backup first.
