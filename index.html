🔄 Convert to Permanent Token
Paste the short-lived token from Graph API Explorer here. Click Convert — the app exchanges it server-side for a permanent Page Access Token that never expires, then saves it automatically.
📡 Facebook Webhook Subscription
Once the webhook is set up in Meta Developer portal, click below to subscribe each page. This tells Facebook to send real-time comments, reactions, and inquiries to the app. Only needs to be done once per page.
About Comments & Auto-Reply
This app does not auto-reply to Facebook comments or DMs. Doing that safely requires a server running continuously to watch for new messages — not something a browser-only app can do, and risky for B2B trust if a bot replies with the wrong info. Instead, the Dashboard's Inquiry & Comment Check reminds you which published posts to look at on Facebook directly, so real inquiries get a real reply from you.
Telegram Bot
Used to notify staff when a post needs their attention. Free, no approval process — create a bot via Telegram's @BotFather, then paste its token here.
To create one: open Telegram, search for @BotFather, send /newbot, follow the prompts, then copy the token it gives you here.
Staff
Each staff member needs to message your bot once on Telegram first (search for it by username and send any message), so the bot can message them back. Then add them here with their Chat ID — they can get this by messaging @userinfobot on Telegram, which replies with their ID. Check which companies they're responsible for, and they'll only see posts from those companies.
